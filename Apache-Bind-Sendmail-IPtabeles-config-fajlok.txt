apt-get install iptables
Nézze meg, hogy vannak-e már szabályok a tűzfalban!
 - iptables -L -v
Dobja el a localhost címre érkező ping (ICMP) csomagokat a tűzfal segítségével!
 - iptables -A INPUT -p icmp -j DROP 
Engedélyezze az SSH kapcsolatot.
- iptables -A INPUT -p tcp --dport 22 -j ACCEPT
Engedélyezze a webszervert.
- iptables -A INPUT -p tcp --dport 80 -j ACCEPT
vagy
- iptables -A INPUT -p tcp -m tcp --dport 80 -j ACCEPT
Engedélyezze a DNS szervert.
- iptables -A OUTPUT -p udp --dport 53 -j ACCEPT
- iptables -A OUTPUT -p tcp --dport 53 -j ACCEPT

Minden titlva
iptables -P INPUT DROP
iptables -P OUTPUT DROP
iptables -P FORWARD DROP

-A kapcsoló: a szabályt a lánc végére szúrja be
-I kapcsoló: a szabályt a lánc elejére szúrja be
INPUT: a gépre bejövő lánc
OUTPUT: a gépről kimenő lánc
FORWARD: hálózatról jövő, de nem a gépnek szánt csomag
-s kapcsoló: IP cím, a küldő IP címe
-d kapcsoló: IP cím, a címzett IP címe
-p kapcsoló: protokoll: tcp, udp, icmp
-m kapcsoló: modul
--sport kapcsoló: küldő port
--dport kapcsoló: a címzett portja
-j kapcsoló: megajda, mit kell tenni:
ACCEPT - fogadás
REJECT - visszadobás és jelzés a küldőnek
DROP - eldobás
LOG - naplózás
